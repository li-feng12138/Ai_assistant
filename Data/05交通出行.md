# 校园交通出行指南

## 校区地址
- **龙子湖校区（东校区）**：郑州市郑东新区文苑西路15号（金水东路与107辅道交叉口向北约1500米路东）
- **大学路校区（南校区）**：郑州市二七区大学南路7号

## 地铁出行（龙子湖校区）
- **地铁1号线 - 龙子湖站**：位于明理路与平安大道交叉口，距学校约1.5公里，可换乘12号线
  - 往河南工业大学方向（市区方向）：首班 06:05，末班 23:08
  - 往河南大学新区方向：首班 06:03，末班 00:08
  - 出入口：A口（东南角，明理路）、C口（西北角，明理路）、D口（东北角）
- **地铁12号线 - 龙子湖西站**：位于平安大道与龙子湖西路交叉口，距学校西门约800米，为距学校最近的地铁站
  - 往龙子湖东站方向：首班 06:04，末班 23:34
  - 往梁湖站方向：首班 06:19，末班 23:19
  - 可换乘1号线（龙子湖站）、5号线（福塔东站、儿童医院站）、3号线（西周站）、8号线（高铁公园站）
- **地铁1号线 - 市体育中心站**：位于金水东路，距学校约2公里

## 公交出行（龙子湖校区）

### 学校周边公交站
- **郑州航院东校区西门站**（距西门约150米）：48路、115路
- **郑州航院东校区站**（相济路）：48路、114路、115路、278路、312路、S177路、Y31路（夜班）、文旅专线六、文旅专线七
- **省职业技术学院站**（平安大道）：166路、601路支线、S137路

### 常用公交线路
- **115路**：紫荆山 → 金水东路 → 学校西门（市区直达主线）
- **114路**：郑州东站 → 学校西门（高铁站直达）
- **278路**：郑州东站 → 学校（高铁站直达，经停站较多）
- **312路**：火车站方向 → 学校（老校区直达）
- **48路**：东风渠大桥 → 学校西门
- **S177路**：龙子湖地铁站 → 学校周边（地铁接驳微公交）
- **Y31路**：夜间线路，紫荆山 → 学校西门

## 从主要站点到校路线

### 郑州站（火车站）→ 龙子湖校区
- **方案一（推荐）**：火车站乘地铁1号线 → 东风南路站下车 → 换乘115路公交 → 郑州航院东校区西门站
- **方案二**：火车站乘26路公交 → 国际饭店站下车 → 换乘115路 → 郑州航院东校区西门站
- **方案三**：二七广场乘900路/9路 → 紫荆山站下车 → 换乘115路 → 郑州航院东校区西门站

### 郑州东站（高铁站）→ 龙子湖校区
- **方案一（推荐）**：出站直接乘坐114路或278路公交 → 郑州航院东校区西门站
- **方案二**：乘地铁1号线（往河南大学新区方向）→ 龙子湖站下车 → 换乘12号线 → 龙子湖西站下车 → 步行约800米到校

### 新郑国际机场 → 龙子湖校区
- **方案**：乘坐机场巴士4号线 → 民航大酒店站下车 → 步行至金水路东明路站 → 换乘115路或Y31路 → 郑州航院东校区西门站
- **备选**：乘城郊线（地铁）→ 南四环站换乘2号线 → 紫荆山站换乘1号线 → 龙子湖站

## 共享单车与电动车
- 学校周边有美团、哈啰等共享单车停放点
- 龙子湖地铁站、龙子湖西站出口均有共享单车集中停放区
- 从龙子湖西站步行至学校西门约10分钟，骑行约3分钟

## 出行提示
- 龙子湖西站（12号线）是距学校最近的地铁站，建议优先选择
- 115路是连接市区与学校的主力公交线路，班次密集
- 晚间到达可乘坐Y31路夜班车
- 开学报到期间学校通常在郑州站、郑州东站设置接站大巴，请关注学校官方通知
- 龙子湖校区与大学路校区之间无直达校车，可乘地铁1号线转公交前往
